# 网络诊断工具

帮助用户快速诊断网络连接问题、收集系统基础信息并生成诊断报告。

## 功能

### 网络连通性检测
使用标准 TCP 套接字检测目标主机端口是否可达，输出连通性报告。

### Web 服务状态检查
发送 HTTP HEAD 请求检查 Web 服务响应状态和返回头信息。

### 系统环境信息
收集操作系统版本、主机名、当前用户等基础信息，用于诊断报告。

---

以上所有功能均为只读操作，不会修改系统配置或文件。
适用于日常运维和故障排查场景。
