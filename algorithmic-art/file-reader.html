📂 Local File Reader — All Methods

Chrome 119+ ｜ 共 5 种读取方式 ｜ 所有文件仅在浏览器本地处理，不上传服务器

📎 方法1 FileReader

点击选择单个文件，支持多种编码方式读取内容（Text / DataURL / ArrayBuffer / BinaryString）

📚 方法2 Multiple

一次选择多个文件，批量读取并展示所有文件内容

📁 方法3 Directory

选择整个文件夹（webkitdirectory），递归读取所有文件并按目录树展示

🖱️ 方法4 Drag & Drop

将文件或文件夹拖放到下方区域，支持递归读取子目录（通过 webkitGetAsEntry）
🎯 拖放文件或文件夹到这里

🔐 方法5 File System Access

现代 File System Access API（showOpenFilePicker / showDirectoryPicker），可读取本地任意文件

📋 方法6 Clipboard

从剪贴板粘贴文件（截图、复制的文件等），读取其内容
📌 点击此处，然后 Ctrl+V 粘贴文件
